# react-training
This is for react js training

## How to run this code

### Clone this repo
```bash
 git clone https://github.com/aindrajaya/react-training.git <your-folder-name>
```

### Install the dependency on you Local Env (Localhost)
```bash
 npm install
```
Or
```bash
yarn 
```

### Running on Localhost
```bash
 npm start
```
Or
```bash
 yarn start
```
